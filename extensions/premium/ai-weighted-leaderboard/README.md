# ai-weighted-leaderboard

**Available exclusively on the ShillJudge hosted platform.**

Replaces the default leaderboard query with an AI-weighted ranking that blends the open-core engagement score with DeepSeek content-quality ratings. Requires `deepseek-review-engine` to have scored threads first.

The manifest here declares the hook interface; the real implementation lives in `shilljudge-premium`.

To use this feature, see [shilljudge.com](https://shilljudge.com) or contact Jird Labs for enterprise licensing.

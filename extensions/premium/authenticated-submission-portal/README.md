# authenticated-submission-portal

**Available exclusively on the ShillJudge hosted platform.**

This extension provides a verified-identity submission portal backed by KYC authentication. It hooks into the thread submission and user login flows to enforce identity verification before contest entries are accepted.

The manifest in this directory declares the extension's interface. The real implementation lives in the private `shilljudge-premium` repository and is loaded only by the licensed hosted platform.

To use this feature, see [shilljudge.com](https://shilljudge.com) or contact Jird Labs for enterprise licensing.
